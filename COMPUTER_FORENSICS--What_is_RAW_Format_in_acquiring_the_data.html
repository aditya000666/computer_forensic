What is RAW Format in acquiring the data


• Not a format
• Chunk of data imaged from evidence source
• Contain no additional metadata aside from information about image file itself (name, size, timestamps and other information in image own inode)
• Extracting raw image is technically transfer of sequence of bytes from source device to destination file
• Done without any transformation of translation
• Disk copying tools such as dd, dc3dd and other variants are used to extract RAW images
