What is a File


• Files have specific structure or format
• Collection of correlated data
• Start of file is called “header”
• End of file is called “footer”
• All of bytes represent the remainder of the file
• File header consists of metadata
• Header is used to identify the file type
• Header is much harder to change and inaccessible to most users
• File is not saved in contiguous clusters sometimes
• Separate pieces of file can be on different sides of platter
• While accessing, computer gets the location of all sectors allocated to file and recreates the file
• To work on the file, it must be loaded into computer RAM
• From RAM, file is loaded into CPU
