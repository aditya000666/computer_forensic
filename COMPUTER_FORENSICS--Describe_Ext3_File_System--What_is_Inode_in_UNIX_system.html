What is Inode in UNIX system


• Each inode correspond to one file and store file primary metadata such as file size, ownership and temporal information
• Inode contains following details
◇ File Type
◇ Permission to that File
◇ Link Count
◇ User ID
◇ Group ID
◇ Size of File
◇ Time Stamp
◇ Attributes
◇ Access Control List
◇ Link to Location of File
◇ Other Metadata
