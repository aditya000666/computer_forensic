NTFS Structure


• Master File Table (MFT) is heart of NTFS containing information about all files and directories
• Every file and directory has at least one entry in table and entries are simple
• Entries are 1KB in size but only first 42 bytes have defined purpose
• Remaining bytes store attributes which are small data structures used for specific purpose
• Microsoft calls each entry in a table a File Record (MFT Entry)
• Each entry is given address based on location in table starting with 0
