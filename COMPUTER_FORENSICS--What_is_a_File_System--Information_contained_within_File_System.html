Information contained within File System


• General File system information i.e. general structure of file system, location of certain data structure and size of data unit
• Contents of the file, organised into collection of standard sized containers like clusters, blocks
• Metadata of the file, FAT directory entries, NTFS MFT entries, UFS and Ext3 inode structures are examples of this type of data structure
• File name
• User quota statistics and file system journals
