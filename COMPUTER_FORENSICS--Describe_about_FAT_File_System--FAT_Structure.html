FAT Structure


• File or Directory Content is stored in data units called clusters
• If file or directory has allocated more than one cluster, than other clusters are found by using structure called FAT structure
• The FAT structure is used to identify the next cluster in file and also used to identify the allocation status of clusters
• Therefore FAT Structure is used in both content and metadata categories
• Three different version of FAT: FAT12, FAT16, FAT32
• Major difference is size of version in FAT
