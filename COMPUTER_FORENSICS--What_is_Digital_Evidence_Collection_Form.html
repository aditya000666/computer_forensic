What is Digital Evidence Collection Form


• Most important element of forensic process
• Necessary steps taken for collection should be accurate and repeatable with same results every time is done
• Should contain information about kind of software and version used and time when collection process started and ended

• Standard details captured in DEC
• Crime Number
• Application Section
• Date
• Name of Investigation Officer
• Address
• System Information
• Type
• Manufacturer
• Model Number
• Serial Number
• BIOS Date/Time
• Evidence Drive Information
• Type of Media
• Hard Disk Type
• Manufacturer
• Model Number
• Serial Number
• Sectors Imaged/Number of Logical Partitions
• General Information
• Software and Version Number
• Write-protect device type
• Drives information
