File Hashing


• Act of attempting to identify file
• Different hash algorithms can be used for this purpose
• Some algorithms are weak and likely to have collisions
• Once file is hashed, can be compared to other file hashes
• Common technique is IR to search for file hash on Google and determine the file
• Match file hash with contraband images to determine file
• Check file for abusive material, identify file by hash value
• Can also be used to whitelist files based on signature
• Whitelisting file result in data reduction in case by removing legitimate files from timeline
