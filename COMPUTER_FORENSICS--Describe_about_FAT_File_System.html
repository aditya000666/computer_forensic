Describe about FAT File System


• File Allocation Table is one of most simple file systems
• Primary file system of Microsoft DOS and Windows 9x series OS but not NT, 2000 and XP defaulted to NTFS
• Supported by all Windows and UNIX OS
• Frequently found in Compact flash cards for Digital cameras and USB
• Basic concept of FAT File System is that each file and directory is allocated data structure, called directory entry that contains all file name, size, starting address of file content and other metadata
