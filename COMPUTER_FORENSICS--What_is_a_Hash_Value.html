What is a Hash Value


• Hash is unique value generated by cryptographic hashing algorithms
• Hash values are used in variety of ways including cryptography and evidence integrity
• Referred as Digital Fingerprints or Digital DNA
• Any change to hard drive, even single bit will result in different hash value
• Any tampering or manipulation of evidence is readily detectable
• Most common hash functions: MD5, SHA1, SHA2
• Used throughout the Forensic process
• Use for verification of cloned evidence and is exact duplicate
• Used as integrity check at any point
• Examiner may change evidence to opposite side for which the hash value is sent along to be compared to original
• Comparison verifies that image is bit for bit copy
• Relevant hash values were generated and recorded throughout the case
• Also included with Final report
• Digital fingerprints are crucial to demonstrate integrity
