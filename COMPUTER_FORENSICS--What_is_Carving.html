What is Carving


• Act of searching for particular strings or bytes within structure
• Hex editor or other data-viewing tools can be used to carve for data
• Analyst determines string or binary pattern to search
• Then initiate search across device or structure for that string or pattern
• Target can be of any scope i.e. file, slack space, unallocated space, full volume, memory image or swap file
• Can be important of finding evidence when deleted
