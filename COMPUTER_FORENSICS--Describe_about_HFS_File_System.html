Describe about HFS File System


• HFS Volume format for MAC OS
• HFS Plus introduced for Mac OS 8.1
• Architecturally similar to HFS
• HFS Plus is specification of how a volume exists on disk
• Managed the files that contain user data, along with structure to retrieve the data
• The storage space on disk is divided into units called sectors
• Sector is smallest part of disk that disk driver software will read or write in single operation
• Size of sector is based on way of data physically laid on disk
• For hard disks, sector is 512 bytes
• For optical media, sector is 2048 bytes
• Data structure on HFS Plus does not depend on size of structure with exception of journal
• Journal rely on accessing individual sectors, sector size is stored in jhdr_size field of journal header (if volume has journal)
• HFS Plus allocates space in units allocation blocks which is group of consecutive bytes
• Size of allocation block is power of 2, greater than or equal to 512
• Value cannot be changed until volume in reinitialised
• Allocation blocks are identified by 32 bit allocation block number
• Current implementation of file system are optimised for 4K
