Slack Space


Slack Space: difference in space assigned and space used. Sectors containing leftover data are referred to slack space or cluster slack

Example
• If created one byte file, NTFS would write single byte and 511 null bytes to cluster
• If cluster size equal to one sector it would be written in same way
• If cluster size was 4Kb i.e. 8 clusters
• Remaining size would not be written to
