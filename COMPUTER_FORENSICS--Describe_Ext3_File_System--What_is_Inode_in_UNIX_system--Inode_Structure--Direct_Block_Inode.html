Direct Block Inode


• Ext2 consist 15 block pointers
• First 12 are called Direct Block pointers
• These pointer point to address of block containing the data of the file
• 12 block pointers can point to 12 data blocks
• Direct block pointer address only 48K (12 * 4K) of data
• If file is of only 48K or below in size, inode itself can address all block containing the data of file
