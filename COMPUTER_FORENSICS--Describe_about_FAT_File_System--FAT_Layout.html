FAT Layout


• First section is Reserved area, and includes data in file system category.
• FAT12 and FAT16 in this area is typically only 1 sector in size but size is defined in boot sector
• Second section is FAT area and contains the primary and backup FAT structure.
• It starts in sector following the reserved area and size is calculated based on number and size of FAT structures
• Third section is Data area, where the clusters that will be allocated to store file and directory content
• In a FAT File System, the file system category of data can be found In boot sector data structure
• Boot sector is located in first sector of volume and is part of reserved area of file system
• There is no field that identifies the file system as FAT12, FAT16 & FAT32
• Type can be determined by performing calculations from boot sector data
• FAT32 file system boot sector contains additional data including the sector address of backup copy of boot sector and major and minor version number
• The backup copy of boot sector can be used if version in sector 0 becomes corrupt, and Microsoft documentation says that it should always be in sector 6 that tools can automatically find it if the default copy goes bad
• FAT32 file systems also have FSINFO data structure that contains information about location of next available cluster and total amount of free clusters
